# 🚨 Smart Emergency Response System

A real-time crisis detection and response prototype for hospitality environments.

## 🔥 Features
- Panic Button Trigger
- Real-time updates using Socket.IO
- Live dashboard status
- Emergency notifications

## 🛠️ Tech Stack
- Flutter (Frontend)
- Node.js + Express (Backend)
- Socket.IO (Real-time communication)

## 🚀 How to Run

### Backend
cd backend
node server.js

### Frontend
cd frontend_flutter
flutter run
